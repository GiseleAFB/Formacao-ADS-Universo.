# 🎓 Formação em Análise e Desenvolvimento de Sistemas - UNIVERSO

Este repositório é o meu portfólio acadêmico, onde registro minha evolução técnica e prática durante a graduação na **Universidade Salgado de Oliveira**. Aqui apresento projetos que demonstram minha capacidade de aplicar lógica de programação, estruturar dados e modelar sistemas reais.

## 🚀 O que você encontrará aqui?

O repositório está organizado por competências:

### 📂 Analise_e_Projeto_Sistemas
Documentação teórica e modelagem de software baseada em UML:
- **Engenharia de Software:** Análise de processos de desenvolvimento (Cascata e Iterativo).
- **UML:** Estudo de mecanismos gerais e Diagramas de Casos de Uso para representação de funcionalidades.

### 📂 Banco_de_Dados_Hospitalar
Projeto completo de gerenciamento de um Centro Cirúrgico:
- **Modelagem Relacional:** Diagramas EER para controle de médicos, salas e pacientes.
- **SQL:** Scripts de criação de tabelas e consultas estratégicas (JOINs).

### 📂 Estrutura_de_Dados_python
Implementação de algoritmos clássicos utilizando estruturas dinâmicas:
- **Pilha (LIFO):** Controle de empilhamento de caixas em depósito.
- **Integração Pilha/Fila:** Sistema escolar para cadastro de alunos e notas.

### 📂 Logica de Programação C
Base em lógica estruturada e algoritmos fundamentais:
- **Cálculos Financeiros:** Automação de folha de pagamento com descontos de INSS.
- **Matemática e Lógica:** Verificador de números primos e resoluções de equações de 2º grau.
